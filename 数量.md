##### 最值问题

1. 多集合反向构造

   - 特征：关键词：至少...至多，都...至少
   - 思路：
     - 问题转换：都...至少 -> 至少一个不的最多...
     - 计算每一个不的情况
     - 将每一个不的情况加起来（因为要求最多嘛，所以极端情况就是没有重复不的）
     - 总的 - 至少一个不的（最多）= 都.. 最少
   - 方法：考虑反面
   - 例1：某社团共有46人，其中35人爱好A，30人爱好B，38人爱好C，40人爱好D；问：这个社团至少有多少人以上四项活动都喜欢？
     - 4项都喜欢的最少->4项都喜欢的 反面：至少一项不喜欢
     - 整个反面 -> 至少一项不喜欢的要最多
     - 不爱好A的有46 - 35 = 11人
     - 不爱好B的有46 - 30 = 16人
     - 不爱好C的有46 - 38 = 8人
     - 不爱好D的有46 - 40 = 6人
     - 因此至少一项不喜欢的最多（不重复情况下最多）有11 + 16 + 8 + 6 = 41人
     - 4项都喜欢的最少有 46 - 41 = 5人
   - 例2：共有100个人参加某公司的招聘考试，考试的内容共有5道题，一至五题分别有80、95、91、82、62人答对，答对3道和3道以上的人员能通过考试
   - 请问至多有多少人能通过考试？
     - 答对5题的最多min(80, 95, 91, 82, 62) = 62人
     - 答对4题的对多min(80-62, 95-62, 91-62, 82-62) = min(18, 33, 29, 20) = 18
     - 答对3题的最多min(33-18, 29-18,20-18) = min(15,11,2) = 2
     - 考试通过的最多62 + 18 + 2 = 82人
   - 请问至少有多少人能通过考试？
     - 做错第一题的有100 - 80 = 20
     - 做错第二题的有100 - 95 = 5
     - 做错第三题的有100 - 91 = 9
     - 做错第四题的有100 - 82 = 18
     - 做错第五题的有100 - 62 = 38
     - 一共做错了 20 + 5 + 9 + 18 + 38 = 90题
     - 问至少多少人通过考试->就是问最多多少人答错3道题->90/3=30
     - 至少100-30 = 70能通过考试

2. 构造数列

   - 特征：关键词：最…最…，排名第几…最…

   - 思路：排序 -> 定位 -> 构造 -> 求和

   - 方法：

     - 求谁设谁，设X
     - 反向推其他
     - 加和求解

   - 例：5人体重之和是423，已知每个人体重都是整数，并且各不相同，则最轻的人，最重可能是多少？

     - 问最轻的最重是多少，那就是比他重的人尽量轻，即跟他差距要小
     - 设他为X，其余X+1、X+2、X+3、X+4
     - 求和：5X+10 = 423->X=82.3->整数=>82

3. 最不利构造

   - 特征：关键词：至少…保证
   - 方法：最坏的情况+1
   - 例：大盒子中有10个红球、8个白球、4个黑球，问，至少摸几次才能保证至少有一个白球？
     - 最坏情况，前几次都不是白球，直到红黑球摸完
     - 10 + 4 + 1 = 15

5. 基期与现期
   - 基期量
     - 识别
       - 求前面某个时期的量
       - 一般在第一题，会根据现期量和增长率=>基期量
     - 公式
       - 【基期=现期-增长量】
       - 【基期=现期÷(1+r)】
       - 记住一点：基期量要小，所以用减法/除法，带符号计算
     - 速算
       - |r|＞5%：截位直除
       - |r|≤5%：化除为乘
         - 【A÷(1+r)≈A×(1-r)】
         - 【A÷(1-r)≈A×(1+r)】
         - 化除为乘一般用在求基期量，而且值比真实值偏小
   - 现期量
     - 识别
       - 求后面某个时期的量
       - 现期量 和 增长率、增长量都是现在的值，比如2021年的产量为16万吨，同比增长10%，增长量为1.4万吨=>说的是2021年产量为16万吨，2021年增长率为10%，2021年的增长量为1.4万吨
     - 公式
       - 【现期=基期+增长量】
       - 【现期=基期×(1+r)】
     - 速算
       - 截位计算
       - 特殊数字
6. 增长率
   - 计算类
     - 识别
       - 增速/增幅/增长百分数/增长几成/增长几倍/增长速度/增长幅度
     - 给百分数
       - 加减计算：2021年增长率为10%，比2020年下降了5个百分点，求2020年增长率=>10% + 5%
     - 给具体量
       - 【r=增长量÷基期量】
     - 增长率与倍数
       - r = n -1
       - n是现期量与基期量倍数，即n=现期量/基期量
   - 比较类
     - 识别
       - 增长最快/增长最慢/增长率最高/增长率最低
     - 倍数关系明显
       - 【现期÷基期=2+】
       - 【现期量÷基期量】
     - 倍数关系不明显
       - 【现期÷基期=1+】
       - 【增长量÷基期量】
     - 使用之前分数比较技巧
   - 百分点和百分数
     - 百分数：表示两个量的比例关系，用除法计算
     - 百分点：表示百分数的变化，用加减法计算
7. 特殊增长率
   - 间隔增长率
     - 识别
       - 中间隔一年求增长率
       - 如：2021年增长率为10%，比上年同期降低5个百分点，求2021相对于2019年的增长率为？r间隔 = 10% + (10% + 5%) + 10%*5%
     - 公式
       - 【r=r1+r2+r1×r2】
     - 速算
       - |r1|≤10%，|r2|≤10%，可忽略r1×r2
       - 百化分
   - 年均增长率
     - 识别
       - 年均增长最快/年均增速排序
     - 公式
       - A = B(1+r)^n => r = (n)√(A/B) - 1 ≈ (A-B)/nB
       - 年均增长率比较，若n相同，只需比较A/B
     - 比较方法
       - 现期÷基期
       - 求n年内几个量年均增长率的大小关系，不需要一个个求，直接比较现期/基期
   - 混合增长率
     - 识别
       - 部分增速与整体增速之间的关系
       - 场景
         - 房产、地产=>房地产
         - 城镇、农村=>全国
         - 上半年、下半年=>全年
         - 税收、其他=>财政总收入
       - 整体增长率 肯定是 介于 各个部分增长率之间的
     - 口诀
       - 居中但不中
         - 2021上半年同比增速为8%，下半年同比增速为12%=>2021年全年的同比增速范围：8%-12%
       - 偏向基期较大的
         - 若2021上半年产量为100，下半年产量为80 => 2021年全年的同比增速：8%-12%并且靠近8%
     - 精确计算
       - 增长率差比 = 基期量的反比，考试中用现期量代替基期量
       - (r整体 - r小)/(r大-r整体) = 大部分的基期量/小部分的基期量 
   - 年份差澄清
     - 2011-2015 江苏：基期2010，现期2015，年份差5年 =>年份差 = 2015 - 2011 + 1
8. 增长量
   - 计算类
     - 识别
       - 增长/单位
     - 公式
       - 【现期÷(1+r)×r】
     - 速算
       - 百化分计算
         - 近似转化
         - 倍数转化
         - 取中转化
       - |r|≈1÷n
         - 【增长量=现期÷(n+1)】
         - 【减少量=现期÷(n-1)】
         - 注意：r要带符号计算
   - 比较类
     - 识别
       - 增长最多/增长最少
     - 速算
       - 给出每年数据
         - 直接两两相减
         - 柱状图还可用直尺标注，可以看斜率
         - 注意：只有比较增长量时，才可以看斜率，当然也可以看高度差
       - 给现期和r
         - 两者都大则增量必大
           - 增长型：现期大、增长速度快 => 增长量大
           - 下降型：现期大、下降速度快 => 减少量大
         - 百化分计算
           - 增长量 = 现期/(1+n)，现期大，增长率大=> 增长量大
9. 比重
   - 现期比重
     - 识别
       - 问题时间与材料时间一致/占/比重
     - 公式
       - 【比重=部分÷总体】
       - 【部分=总体×比重】
       - 【总体=部分÷比重】
     - 速算
       - 截位直除
   - 基期比重
     - 识别
       - 问题时间在材料时间前/占/比重
     - 公式
       - 【A÷B×(1+b)÷(1+a)】
         - 如果整体的增长率小于部分增长率，则现期比重 > 基期比重
         - 如果整体的增长率大于部分增长率，则现期比重 < 基期比重
         - 因为比重是指 部分/整体，相比基期，部分增长快于整体增长，说明现期比重大于基期比重
     - 速算
       - 截位直除
       - 先计算现期比重，再判断大小
         - 根据公式，先计算现期比重，然后根据(1+b)/(1+a)的值判断选项
   - 两期比重
     - 识别
       - 题干涉及两个时间/比重，现期比重 - 基期比重
     - 公式
       - A/B - [A(1+b)]/[B(1+a)] = A÷B×(a-b)÷(1+a)
         - a>b => 比重上升
         - a<b => 比重下降
         - 注意：如果要计算出两期比重上升/下降的百分点，先计算现期比重
     - 速算
       - 比较部分与总体的增长率，部分大则提升，总体大则降低
       - 两期比重差一定小于增长率差
         - A/B - [A(1+b)]/[B(1+a)] = A÷B×(a-b)÷(1+a) < a-b
   - 增长贡献率
     - 部分增量在整体增量中所占比重
     - A增长量为100，整体增长量为1000，那么A对整体的增长贡献率为：100/1000 = 10%
   - 拉动增长率
     - 部分增量在整体基期量中所占比重
     - A增长量为100，整体基期量为10000，那么A对整体的拉动增长率为：100/10000 = 1%
10. 平均数
    - 现期平均
      - 识别
        - 问题时间与材料时间一致/平均
      - 公式
        - 【平均数=总数÷个数】
      - 速算
        - 截位直除
        - 削峰填谷
          - 求和的几个数比较大，参差不齐，可以找一个整一点的数作为基准，算差距的平均值。再加上基准
    - 基期平均
      - 识别
        - 问题时间在材料时间前/平均
          - 求上一年的平均数
      - 公式
        - 【A÷B×(1+b)÷(1+a)】
        - 形式和基期比重公式一样
      - 速算
        - 截位直除
        - 先计算现期平均，再判断大小
    - 两期平均
      - 识别
        - 题干涉及两个时间/平均
      - 公式
        - A/B - [A(1+b)]/[B(1+a)] = A÷B×(a-b)÷(1+a)
        - 【(a-b)÷(1+b)】
      - 速算
        - 截位直除
        - 比较分子与分母的增长率，分子大则提升，分母大则降低
11. 倍数
    - 现期倍数
      - 识别
        - 问题时间与材料时间一致/A是B的多少倍
      - 公式
        - 【A÷B】
      - 速算
        - 截位直除
    - 基期倍数
      - 识别
        - 问题时间在材料时间前/A是B的多少倍
      - 公式
        - 【A÷B×(1+b)÷(1+a)】
        - 形式和基期比重、基期平均数公式一样
      - 速算
        - 截位直除
        - 先计算现期倍数，再判断大小
      - 增量倍数
        - 识别
          - A的增量是B的多少倍
        - 公式
          - A\*a/(1+a) / [B\*b/(1+b)] = A/(1+n1) / [B/(1+n2)] = A/B\*(1+n2)/(1+n1)
    - 倍数与增长
      - A是B的n倍:【n=A÷B】
      - A比B多n倍:【n=A÷B-1】
